# AI-Based Credit Risk Prediction System

## Overview
This project predicts customer credit risk using Machine Learning models trained on the German Credit Dataset.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Logistic Regression
- Decision Tree
- Random Forest
- Customer Segmentation using K-Means
- Model Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Dataset
German Credit Dataset
